# Locale Tester

Test and compare locale behavior on Linux systems.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install locale-tester
```

### Fedora/RPM
```bash
sudo dnf install locale-tester
```
